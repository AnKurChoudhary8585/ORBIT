# 🚀 Orbit - Video Hosting & Sharing Platform

https://orbit-video-platform.netlify.app/
Orbit is a modern video hosting and sharing platform built using the MERN Stack. It allows users to upload, store, manage, and share videos seamlessly. The platform leverages Cloudinary for media storage and delivery, ensuring optimized performance and scalability.

## ✨ Features

- 🔐 User Authentication & Authorization
- 📹 Video Upload and Streaming
- 🖼️ Thumbnail/Image Upload Support
- ☁️ Cloudinary Integration for Media Storage
- 🗄️ MongoDB Database for Persistent Data Storage
- 👤 User Profiles and Video Management
- ❤️ Like and Engagement System
- 🔍 Video Discovery and Search
- 📱 Responsive User Interface
- ⚡ Fast and Scalable MERN Architecture
